# 🌱 AgroPlus - Calculadora de Sementes

Esse projeto é uma ferramenta digital para auxiliar produtores agrícolas a calcularem o custo de sementes com base na área de plantio!

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica.
- **CSS3**: Design responsivo e acessível.
- **JavaScript**: Lógica de cálculo e interatividade.
(negócio tosco kk)

## 🛠️ Como usar
1. Informe a área total do seu terreno em m2
2. Insira o preço unitário da semente
3. Clique em "Calcular" para ver o investimento total

## ⚖️ Ética e Dados
Projeto desenvolvido simplesmente, com foco em acessibilidade visual e baseado em dados de produtividade simulados!!!
